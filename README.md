# Simple Todo App

This is an application that allows the user to create, update, and delete a simple To do List; includes an optional due date to allow users to organize tasks in order of importance.

## Functionality

This application was created so users could have an easy way to track important tasks; perfect for students, clerical clerks and other users needing to organize simple daily tasks. The app allows users to create new tasks and has the function to check each task as done once completed. An optional Due Date application is available so users can easily see what tasks are needed to be completed first and when. The app also allows users to delete tasks if no longer applicable.

## Technology

Technologies, used are HTML5, CSS, JavaScript.
